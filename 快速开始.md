# LangChain4j + Spring Boot 快速开始

## 🚀 5 分钟快速配置

### 步骤 1：获取阿里云 API Key

1. 访问：https://dashscope.console.aliyun.com/
2. 登录阿里云账号
3. 开通 DashScope 服务（免费额度可用）
4. 创建 API Key
5. 复制 API Key（格式：`sk-xxxxxxxxxxxxxxxxxxxxxxxx`）

### 步骤 2：配置 API Key

**方式一：环境变量（推荐）**
```bash
# Windows PowerShell
$env:DASHSCOPE_API_KEY="sk-你的 API Key"

# Linux/Mac
export DASHSCOPE_API_KEY="sk-你的 API Key"
```

**方式二：修改配置文件**
编辑 `power-trade-rag-api/src/main/resources/application.yml`：
```yaml
langchain4j:
  dashscope:
    api-key: sk-你的 API Key  # 替换这里
```

### 步骤 3：编译项目

```bash
cd d:\project2\power-trade-rag
mvn clean install -DskipTests
```

### 步骤 4：启动服务

**启动后端：**
```bash
cd power-trade-rag-api
mvn spring-boot:run
```

**启动前端（新终端）：**
```bash
cd power-trade-rag-web
npm run dev
```

### 步骤 5：访问系统

- 🌐 前端界面：http://localhost:3001/
- 🔌 API 服务：http://localhost:8080/
- 📖 API 文档：http://localhost:8080/swagger-ui.html

## ⚙️ 配置说明

### 模型选择

```yaml
langchain4j:
  dashscope:
    api-key: sk-xxx
    embedding-model-name: text-embedding-v2  # 嵌入模型
    language-model-name: qwen-plus           # 语言模型
```

**可用模型：**
- **语言模型**：qwen-turbo、qwen-plus、qwen-max
- **嵌入模型**：text-embedding-v1、text-embedding-v2

**推荐配置：**
- 开发测试：qwen-turbo（便宜、快速）
- 生产环境：qwen-plus（平衡性能）
- 高要求场景：qwen-max（最强性能）

### 向量存储

**当前配置：内存存储**
```java
@Bean
public EmbeddingStore<TextSegment> embeddingStore() {
    return new InMemoryEmbeddingStore<>();
}
```

**优点：**
- ✅ 零配置
- ✅ 快速启动
- ✅ 适合开发测试

**缺点：**
- ❌ 重启后数据丢失
- ❌ 不支持分布式
- ❌ 内存限制

**生产环境建议升级：**
- ChromaDB
- Milvus
- Elasticsearch

## 💡 使用技巧

### 1. 文档预处理

上传文档前建议：
- 确保文档内容清晰
- 避免扫描版 PDF（OCR 识别率低）
- 使用结构化文档（标题、段落分明）

### 2. 优化问答质量

**提示词优化：**
```java
private String buildPrompt(String query, String context) {
    return "您是一位电力交易领域的专业助手...\n\n" +
           "请根据参考资料回答...\n\n" +
           "如果资料中没有相关信息，请如实告知。";
}
```

### 3. 调整分块参数

```yaml
rag:
  document:
    chunk-size: 300      # 分块大小（200-500）
    chunk-overlap: 30    # 重叠部分（20-50）
```

**建议：**
- 技术文档：chunk-size=300
- 政策法规：chunk-size=400
- 合同协议：chunk-size=500

## 🐛 常见问题

### Q1: API Key 错误

**错误信息：**
```
Invalid API Key
```

**解决：**
1. 检查 API Key 是否正确复制
2. 确认已开通 DashScope 服务
3. 检查 API Key 是否过期

### Q2: 依赖下载失败

**错误信息：**
```
Could not resolve dependencies
```

**解决：**
```bash
# 清理本地仓库
rm -rf ~/.m2/repository/dev/langchain4j

# 重新编译
mvn clean install -U
```

### Q3: 内存溢出

**错误信息：**
```
OutOfMemoryError
```

**解决：**
```bash
# 增加 JVM 内存
export MAVEN_OPTS="-Xmx2g"
mvn spring-boot:run
```
