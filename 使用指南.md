# 电力交易智能问答系统 - 可视化知识库管理使用指南

## 🎯 功能概述

本系统提供**智能问答**和**可视化知识库管理**两大核心功能，两者位于同一层级，可通过左侧导航栏快速切换。

## 📱 界面布局

### 主界面结构
```
┌─────────────────────────────────────┐
│ 侧边栏 │  顶部导航栏                │
│        ├────────────────────────────┤
│ 智能问答 │  主内容区                  │
│ 知识库管理│                          │
│ 文档管理 │                          │
└─────────────────────────────────────┘
```

### 导航菜单
- **智能问答** - 与 AI 机器人对话，获取电力交易相关知识
- **知识库管理** - 可视化管理知识库（创建、编辑、删除、查看文档）
- **文档管理** - 管理所有文档

## 🤖 智能问答使用

### 操作步骤
1. 点击左侧菜单「智能问答」
2. 在输入框中输入问题，例如：
   - 什么是电力中长期交易？
   - 电力现货市场包含哪些部分？
   - 什么是可再生能源电力消纳？
   - 电力交易价格如何确定？
3. 按回车键或点击「发送」按钮
4. 系统会自动检索相关知识并给出智能回答

### 功能特点
- ✅ 支持多轮对话
- ✅ 自动保存会话历史
- ✅ 显示回答时间
- ✅ 美观的聊天气泡样式

## 📚 知识库可视化管理

### 1. 统计概览
进入「知识库管理」页面，顶部显示四个统计卡片：
- 📊 **知识库总数** - 系统中所有知识库数量
- 📁 **文档总数** - 所有知识库中的文档总数
- ✅ **启用中** - 当前启用状态的知识库数量
- 💬 **问答次数** - 系统累计问答次数

### 2. 创建知识库

#### 方法一：通过界面创建
1. 点击「知识库管理」页面右上角的「创建知识库」按钮
2. 在弹出的对话框中填写：
   - **知识库名称**（必填）- 如"电力交易政策法规"
   - **知识库描述**（必填）- 如"包含国家及地方电力交易相关政策法规"
   - **状态** - 启用/禁用
3. 点击「确定」保存

#### 方法二：通过 API 创建
```bash
POST http://localhost:8080/api/knowledge/create
Content-Type: application/json

{
  "name": "电力交易政策法规",
  "description": "包含国家及地方电力交易相关政策法规",
  "status": 1
}
```

### 3. 查看和搜索知识库

#### 搜索功能
- 在搜索框中输入关键词
- 支持按知识库名称或描述搜索
- 实时过滤显示结果

#### 知识库列表信息
列表显示以下内容：
- 知识库 ID
- 知识库名称（带文件夹图标）
- 描述
- 状态（启用/禁用标签）
- 文档数量
- 操作按钮（查看文档、编辑、删除）

### 4. 编辑知识库
1. 找到要编辑的知识库
2. 点击「编辑」按钮
3. 修改名称、描述或状态
4. 点击「确定」保存

### 5. 查看关联文档
1. 点击知识库行的「查看文档」按钮
2. 弹出对话框显示该知识库下的所有文档
3. 文档列表包含：
   - 文档 ID
   - 文档标题
   - 文档类型（POLICY/CASE/RULE）
   - 创建时间
   - 删除按钮

### 6. 删除知识库
1. 点击要删除知识库行的「删除」按钮
2. 确认删除操作
3. 系统会提示删除成功

⚠️ **注意**：删除知识库不会删除关联的文档

## 📄 文档管理

### 上传文档
1. 点击左侧菜单「文档管理」
2. 点击「上传文档」按钮
3. 选择文件并填写信息
4. 选择关联的知识库
5. 点击「确定」上传

### 删除文档
1. 在文档列表中找到要删除的文档
2. 点击「删除」按钮
3. 确认删除

## 🔧 技术特性

### 前端技术栈
- Vue 3 + Composition API
- Element Plus UI 组件库
- Vue Router 路由管理
- Axios HTTP 客户端

### 后端技术栈
- Node.js + Express
- MySQL 数据库
- RESTful API 设计

### API 接口

| 功能 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 智能问答 | POST | /api/chat/ask | 发送问题获取答案 |
| 获取知识库列表 | GET | /api/knowledge/list | 获取所有知识库 |
| 创建知识库 | POST | /api/knowledge/create | 创建新知识库 |
| 更新知识库 | PUT | /api/knowledge/:kbId | 更新知识库信息 |
| 删除知识库 | DELETE | /api/knowledge/:kbId | 删除知识库 |
| 获取文档列表 | GET | /api/document/list | 获取所有文档 |
| 上传文档 | POST | /api/document/upload | 上传新文档 |
| 删除文档 | DELETE | /api/document/:docId | 删除文档 |
| 获取统计数据 | GET | /api/chat/stats | 获取问答统计 |

## 💡 最佳实践

### 知识库组织建议
1. **按业务领域分类**
   - 电力交易政策法规
   - 电力市场交易案例
   - 电力价格机制
   - 新能源交易

2. **文档命名规范**
   - 使用清晰的文档标题
   - 添加详细的文档描述
   - 正确选择文档类型

3. **定期维护**
   - 及时更新过时的政策文档
   - 清理无效知识库
   - 定期检查文档关联关系

### 问答优化建议
1. **提问技巧**
   - 问题要具体明确
   - 使用专业术语
   - 避免模糊表述

2. **知识库扩充**
   - 上传高质量的文档
   - 确保文档内容准确
   - 定期更新知识库
