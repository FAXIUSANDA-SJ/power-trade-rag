## 🔧 技术说明

### 文件处理架构变更
**原设计**：
```
MultipartFile → FileUtil.saveUploadedFile() → DocumentService.processDocument()
```

**新设计**：
```
MultipartFile → Controller.getBytes() → FileUtil.saveFile() → DocumentService.processDocument()
```

**优势**：
- FileUtil 不再依赖 Spring
- common 模块更轻量
- 便于在其他模块复用
