## 🔧 核心功能

### 1. 双向量数据库支持

通过 `@ConditionalOnProperty` 注解实现动态切换：

```java
@Bean
@ConditionalOnProperty(name = "rag.vector-database-type", havingValue = "chroma")
public EmbeddingStore embeddingStoreChroma(...) {
    // ChromaDB 实现
}

@Bean
@ConditionalOnProperty(name = "rag.vector-database-type", havingValue = "milvus")
public EmbeddingStore embeddingStoreMilvus(...) {
    // Milvus 实现
}
```

### 2. OpenAI 嵌入集成

使用 `OpenAiEmbeddingModel` 生成 1536 维向量：

```java
@Bean
public EmbeddingModel embeddingModel() {
    return OpenAiEmbeddingModel.builder()
            .apiKey(apiKey)
            .modelName("text-embedding-ada-002")
            .build();
}
```

### 3. 向量存储管理

完整的 CRUD 操作：

- ✅ 添加文本片段（单个/批量）
- ✅ 相似度搜索（全局/按知识库）
- ✅ 删除向量（按文档/按知识库）
- ✅ 统计信息查询

### 4. RAG 流程优化

完整的检索增强生成流程：

1. 文档预处理 → 分块
2. 向量化 → 存储到 ChromaDB/Milvus
3. 用户查询 → 向量化
4. 相似度检索 → 检索相关文档
5. 构建上下文 → 生成答案

## 🚀 部署方式

### Docker Compose（推荐）

```bash
# 一键启动所有服务
docker-compose -f docker-compose-vector-db.yml up -d

# 包含的服务：
# - ChromaDB (端口 8000)
# - Milvus (端口 19530)
# - etcd (Milvus 依赖)
# - minio (Milvus 依赖)
# - attu (Milvus GUI, 端口 3000)
```

### 单独部署

```bash
# ChromaDB
docker run -d -p 8000:8000 chromadb/chroma:latest

# Milvus（需要 etcd 和 minio，建议使用 docker-compose）
```

## ⚙️ 配置示例

### 使用 ChromaDB

```yaml
rag:
  vector-database-type: chroma
  chromadb:
    host: localhost
    port: 8000
    collection-name: power-trade-knowledge
```

### 使用 Milvus

```yaml
rag:
  vector-database-type: milvus
  milvus:
    host: localhost
    port: 19530
    username: root
    password: Milvus
    collection-name: power_trade_knowledge
```

## 🎯 使用流程

### 1. 启动向量数据库

```bash
docker-compose -f docker-compose-vector-db.yml up -d
```

### 2. 设置 API Key

```bash
export OPENAI_API_KEY="sk-xxxxxxxx"
```

### 3. 修改配置

编辑 `application.yml`，选择向量数据库类型

### 4. 启动应用

```bash
mvn spring-boot:run
```

### 5. 测试验证

```bash
# 上传文档
curl -X POST http://localhost:8080/api/documents/upload \
  -F "file=@test.pdf" \
  -F "kbId=KB001"

# 智能问答
curl -X POST http://localhost:8080/api/chat \
  -H "Content-Type: application/json" \
  -d '{"query":"什么是电力交易？","kbId":"KB001"}'
```

## 📝 注意事项

1. **OpenAI API Key**
   - 需要准备 OpenAI API Key
   - 如无法访问 OpenAI，可使用国内代理服务

2. **网络要求**
   - 需要能访问 OpenAI API
   - 本地 Docker 网络需畅通

3. **资源要求**
   - ChromaDB: 内存 ~500MB
   - Milvus: 内存 ~2GB
   - 磁盘空间根据数据量而定

4. **成本考虑**
   - OpenAI Embedding: $0.0001 / 1K tokens
   - 建议批量处理、缓存结果
